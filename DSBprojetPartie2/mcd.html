Modèle Conceptuel de Données

Explication des éléments

Jeux-vidéos

  1. ID : Clé primaire de l'élément, permet d'identifier les différents jeux même si les noms sont les mêmes.
  2. Nom : Nom attribué du jeu.
  3. Date de sortie : Date de la sortie du jeu, la première version disponible au public si il en existe plusieurs.
  4. Note JVC : Note attribuée au jeu par le site référence jeuxvideos.com.
  5. PEGI : Le système d'évaluation européen des jeux vidéo qui permet de les classer par âges pour informer les consommateurs.
  6. RECOMPENSE : L'identifiant vers la récompense attribuée au jeu aux Game Awards, si il en a reçu.
