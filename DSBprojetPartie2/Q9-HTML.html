Introduction

Bienvenue sur notre site créé pour le projet de DSB de l'année 2023.

Celui-ci porte sur le thème des jeux vidéo. D'abord, rappelons ce qu'est un jeu vidéo. Selon wikipédia, un jeu vidéo est un jeu électronique doté d'une interface utilisateur permettant une interaction humaine ludique en générant un retour visuel sur un dispositif vidéo.
C'est un domaine dans lequel nous sommes très investis et ce site peut vous donner plusieurs informations sur les différents éléments qui entre en jeu lors de sa création.
À l'aide des onglets situés en haut de la page, vous pouvez naviguer entre les différentes catégories.

Note : Je n'ai pas eu le temps de perfectionner l'affichage des pages, vous aurez alors la meilleure expérience uniquement si votre navigateur prend tout votre écran.
